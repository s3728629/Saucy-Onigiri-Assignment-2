Detailed User Stories

Intro Quiz

User Action System Reaction
User is instructed to read the questions and options corresponding to the question. Acknowledges and understands the questions and answers
The user clicks drop down menu for a few of the questions and views the possible options. Then the user clicks the option that they prefer. The drop down menu tab will result to display the option that the user has chosen.
For the other questions, they are in multiple choice format. The user will click on a checkbox that is next to the option they prefer. The checkbox will change it’s appearance to blue.
Once they have completed the quiz, they can click on the ‘Finished’ button. When they have clicked the ‘Finished’ Button, it will take them to the home page of the main website.
Otherwise, the user can skip doing the pre-quiz and press the ‘Skip’ Button’ at the top of the page. Once they have pressed the button, it will navigate them to the home page.

Tutorials

User Action System Reaction
After taking the quiz, the user will be led to the recommended tutorials for them. The user can start tutorials, which they would start with either watching the video or reading instructions. If the user chooses to watch the video first, they would then scroll down to view the instructions. The user will then be met with the option to take a post-quiz to test their knowledge. The user will also have an option to print out a pdf guide of the tutorial. After watching the video, the user will have gained knowledge of how to use a certain feature. If, after watching the video, they are not sure the user can view the step-by-step instructions to further consolidate their knowledge. The user may also choose to take the post quiz and will be led to the post quiz.
The user chooses the ‘How to send images on Messages for iPhone’ tutorial The user is led to the ‘How to send images on Messages for iPhone’ tutorial page.
The user clicks on the play button of the video and follows along on their phone. The video starts playing
After the video is finished, the user scrolls down to view the step-by-step instructions. The step-by-step instructions are shown on the screen.
The user scrolls down and sees two options to either take a post quiz or to print a pdf guide version. The user then chooses to print a pdf. The screen opens up another tab which opens up the pdf where the user can print it..
The user heads back to the previous page and sees the option to take a post quiz. They then click on the post quiz button to do so. User is led to the post-quiz page.

Help / Chatbot

User Action System Reaction
When the user opens the website, they can access the chatbot from the navigation bar should they have any immediate queries or if this is not their first time using the site and they had a query from their previous session.
The user can also access the chatbot from the ‘help’ buttons located after a pre-quiz/post-quiz or tutorials. 		The chatbot will respond to queries using a database of keywords.
Queries like ‘ how to send an email’ or ‘where can i change font size’ will be responded with a direct link to that page/tutorial.
User clicks on ‘Help’ tab on navigation bar Site opens up the page with the chatbot
User clicks on input box and types their query Blinking vertical line follows the massage as it is typed
User clicks on the ‘send’ button to send a message to the chatbot. The user will get an automated response based on the keywords they imputed. The sent message will appear on the bottom right side of the screen above the text area and the chatbot will reply.
If a user asks a question which has a tutorial, the chatbot will send the reply with a link.
Used clicks on link Directs them to that tutorial page
